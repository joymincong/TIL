# Javascript condition and loop

## Condition

### if

**사용 예시**

```
if(비교문1){
  code 1
} else if(비교문2){
  code 2
} else {
  code 3
}
```



## Loop

### while

**사용 예시**

```
while(비교문){
  code 1
}
```



## for

**사용 예시**

```
for(var i = 0(초기값), i < 3(조건), i++(증가)){
  code
}
```

- break : loop 중단 (if 문과 같이 사용)
- continue : 밑에 무시 (if문과 같이 사용)
