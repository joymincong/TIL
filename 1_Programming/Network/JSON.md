# JSON

Javascript Object Noatation



서로 다른 언어들간에 데이터(텍스트)를 주고 받는(Parsing) 방법 (문서)

- 경량
- 데이터 교환 방식(다른 언어끼리 교환이 가능)



#### XML과 비교

- 마크업 형식(태그)로 데이터를 전송하는 방법
- XML은 문법이 복잡해 용량이 커진다는 단점이 있다. 



### 형식

#### Object

`{"id" : "egoing", "pwd" : "1234"}`

#### Array

`[1, 2, 3, 4]`

#### Value

종류

- string, number, number, array, true, false, null 

#### String

`""`안에 표현 

- `/`은 이스케이프 문자

#### number

`-123` : 음수

`1.1111` : 실수

`1.1E10` : 큰수 표현
